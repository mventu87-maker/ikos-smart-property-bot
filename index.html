¿Cómo funciona el prototipo más simple?

El objetivo de esta primera versión es validar la idea con usuarios reales, usando una estructura muy ligera: una pequeña base de datos, ChatGPT como motor de IA y una landing publicada en GitHub Pages.

Paso 1

El cliente envía la referencia

El usuario escribe la referencia del inmueble (por ejemplo, 1826) desde la web o desde un canal como WhatsApp.

Paso 2

Búsqueda en la base de datos

El sistema busca esa referencia en un archivo sencillo (CSV o JSON) con los datos del inmueble: precio, metros, fotos y estado.

Paso 3

Generación de respuesta con IA

ChatGPT transforma los datos en un mensaje claro y estructurado que cualquier cliente pueda entender en segundos.

Paso 4

Envío instantáneo al usuario

La ficha completa se muestra en la web o se envía como mensaje, reduciendo al mínimo la intervención manual del agente.

Beneficios para tus clientes

  • ✓ Transparencia: ven precio, características y disponibilidad en un solo mensaje.

Preguntas frecuentes del prototipo

Esta versión está pensada para validar la idea con usuarios reales antes de integrarla en sistemas más complejos.

¿Necesito conectar ya mi ERP o CRM?

No. En el prototipo más simple, los datos pueden guardarse en un archivo independiente (por ejemplo, un CSV o JSON) con una selección de inmuebles de prueba.

¿Se puede usar solo como landing de demo?

Sí. La landing publicada en GitHub Pages sirve como demostración visual del concepto y como base para mostrar el flujo de información en presentaciones o en un vídeo explicativo.

¿Qué se valida exactamente con este prototipo?

Principalmente, si los usuarios valoran recibir información instantánea por referencia y si esto reduce la carga de trabajo del agente en consultas repetitivas.

¿Se puede extender a WhatsApp u otros canales?

Sí. Una vez validada la idea, el mismo flujo se puede adaptar a un bot de WhatsApp o integrarse en la web principal de la inmobiliaria.
